# 菲涅耳波带片聚焦成像特性研究

第七届同济大学创新物理实验设计竞赛参赛项目

## 项目简介

本项目系统研究菲涅耳波带片的聚焦与成像特性。制作了覆盖三个参数的 **30 片**样品，构成 $5\times3\times2$ 完全因子实验设计，分析各参数的主效应及交互效应。

| 参数 | 取值 |
| --- | --- |
| 环数 $N$ | 20、40、60、80、100 |
| 设计焦距 $f$ | 400 mm、600 mm、800 mm |
| 类型 | 正型（中心透光）、负型（中心不透光） |

激光波长：$\lambda = 650\ \text{nm}$，加工精度：DPI = 3000（像素尺寸约 8.5 μm）

## 快速上手

1. 从 `FresnelZonePlate_*.pdf` 打印所需波带片图样
2. 按照 `波带片实验指导书.md` 搭建光具座装置
3. 以 **N60-F600-正** 为基准样品完成调光流程
4. 依次开展主效应实验（15 片正型）和正/负对比实验（全部 30 片）

## 核心公式

第 $n$ 个分界圆半径：

$$r_n = \sqrt{n\lambda f}$$

最外环宽度（决定加工可行性）：

$$w_N = r_N - r_{N-1} \approx \frac{1}{2}\sqrt{\frac{\lambda f}{N}}$$

聚焦对比度：

$$C = \frac{I_{\max} - I_{\text{bg}}}{I_{\text{bg}}}$$
